Superalink大陆5日套餐购买攻略(附折扣码)

发表于 2026-05-09 分类于 eSIM 阅读次数：

最近在V2EX论坛，有V友问我Superalink怎么用，干脆整理成一篇攻略，方便回顾。
Ref: https://www.v2ex.com/t/1210180

套餐情况

我目前用的是大陆地区 5 日套餐，参数如下：

  • 有效期：5天
  • 流量：每日5GB（其中热点分享上限2GB）
  • 原价：200泰铢（约42元人民币）
  • 折扣价：25泰铢（约5.26元人民币）——需要折扣码，结算时选泰铢货币

    折扣码：ROY000000
    网址： https://www.superalink.com/destination/aff/ROY000000

算下来每个月买6次，月资费约31.5元，可以用150GB境外流量（覆盖HK+SG），日常看视频没问题。

使用前提：你的设备得支持 eSIM

Superalink是eSIM流量卡，购买后会给你一个QR二维码或激活码(LPA 开头的字符串)，用手机导入即可。

两种设备情况：

  1. 手机原生支持eSIM(iPhone XS及以后的国际版、部分安卓旗舰)
    1. 直接扫码激活，没有任何门槛
  2. 手机不支持原生eSIM(比如我用的国行iPhone 13 PRO)
    1. 可以买一张实体eSIM转接卡，比如ESTK.ME的卡，原理是把eSIM profile写进实体卡里，插到手机用。市面上还有”小白卡”也是类似方案，贴牌的多。只要能扫码或者输入LPA激活码就行。
    2. ESTK官网：https://store.estk.me/
    3. 我的邀请码：UPN3ALUL53
    4. 填写邀请码可以抵扣10%OFF




一个必须注意的设置

第一次用大概率会遇到：有信号、没网络。
原因：数据漫游默认是关的。
去「设置 → 蜂窝网络 → 对应eSIM → 数据漫游」，打开就好了。

使用限制说明

  • 这张卡只能在大陆境内用，买的是”大陆地区”套餐，出了大陆就没流量了。
  • 流量走的是移动基站，漫游到海外流量运营商的节点，所以能绕过GFW ，访问Google、YouTube之类的没问题。
  • 每次购买需要用新邮箱，老邮箱折扣码会失效——用Gmail的 + 别名（yourname+001@gmail.com）或者Cloudflare catch-all转发都能搞定。

操作节奏

每5天买一次，熟了以后整个流程3分钟以内。唯一麻烦的就是要换邮箱，不过解决了邮箱问题之后基本就是重复操作了。
